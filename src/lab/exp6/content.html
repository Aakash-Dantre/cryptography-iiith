Cryptographic Hash Functions and Applications(HMAC)
Introduction

SHA-1 is a popular heuristic hash function that is currently in vogue.  In this experiment, we shall familiarize ourselves with SHA-1 as well as look at one important applaication of hashing, namely, the HMAC algorithm which is currently used in the Internet to achieve data integrity

To understand the need,design and applications of collision resistant hash functions.

Manual

STEP 0: Familiarize yourself with the working of SHA-1. Though we would be using a dummy hash in the sequel for simplicity, in general, you could be using SHA-1 instead

STEP 1: Select a plaintext for which the HMAC tag is to be computed.

STEP 2: Select 'l' such that l < (length of plaintext)/4.

STEP 3: Select an Initialization Vector, IV of length l.

STEP 4: Use the ipad and opad as described in theory part to compute the ciphertext with the help of the hash function provided to you. STEP 5: Get the hashed value by providing a string of length 2*l in the field titled 'Your text' and clicking on 'get hash'. Repeat this for as many times as you need to get the encrypted text for the plaintext selected in Step 1.

STEP 6: Write the final cipher text in 'Final Output' field.
